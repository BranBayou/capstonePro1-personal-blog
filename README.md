To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

<ul>
  <li>Vs code</li>
  <li>npm packages</li>
</ul>


### Setup

Clone this repository to your desired folder: git@github.com:BranBayou/Mobile-First-Portfolio-Project.git

1. Clone the repo:
